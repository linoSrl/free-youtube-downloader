# free-youtube-downloader
Application locale gratuite permettant de télécharger des vidéos et de l’audio depuis YouTube. Il suffit de coller un lien et choisir la qualité (jusqu’à 4K ou audio). L’interface est simple, et le traitement se fait en local avec Python, yt-dlp et ffmpeg. Les fichiers sont enregistrés directement dans le dossier Téléchargements de l’utilisateur.
