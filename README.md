# YouTube Downloader Local

A simple local application to download videos and audio from YouTube in high quality (up to 4K).

---

## Features

- Download YouTube videos in multiple qualities (144p → 4K)
- Extract audio only (MP3 format)
- Simple web interface
- Automatic download to the system **Downloads** folder
- Runs locally (no external website required)

---

## Installation

### 1. Install Python
Make sure you have Python 3.10+ installed.
