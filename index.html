Téléchargement YouTube

Interface ultra simple. Colle un lien, choisis la qualité, puis lance le téléchargement. Le fichier sera enregistré dans le dossier Téléchargements via un petit serveur local.

Cette page ne télécharge pas toute seule depuis le navigateur. Elle envoie le lien à un serveur local sur ton ordinateur, et c’est ce serveur qui enregistre le fichier dans Téléchargements.
Serveur attendu : http://localhost:5000/download
